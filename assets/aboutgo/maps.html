Maps

Maps

Much like the common languages, Go offers some form of Hash Table so that it can allow for complex storage but at as close to constant data retrieval and insertion times. Maps are the data structure that essentially means Hash Table. Like channels maps are nil until initalized with the make command. In addition, they are built directly into Go therefore no import is necessary to use them.

Creating a map

You can create a map using the following syntax: m := make(map[KeyVal]ValueType

Here KeyVal represents the key that will be hashed to find the storage location within the map and the ValueType repesents that value that will be associated with that key. Additionally, instead of the local shorthand := they can also be associated to the global var declaration.

Using a map

Using a map is very simple thanks to its built-in methods.

Inserting/Overwriting: mapName[keyVal] = value

Accessing Value of Key: mapName[keyVal]

When accessing a map, there is another option to get data. If you want to also get if the key was present in the map, you can use the optional _, before the variable name. Take for instance _, data := mapName["test"]. If there is no key with the name "test" in this map then data will contain false, otherwise true.

Delete Key/Value: delete(mapName, keyVal)

Immediate Declaration/Initialization: mapName := map[string]int{"Buck": 1}" which creates a new map and adds value 1 at the hashed keyVal "Buck".

Length (number of elements) of Map: len(mapName)
