Pointers and Structs

Structs in Go

Structs in go are quite similar to C. Each struct is a new type with members that can be given values individually (as show in example to right) or all at once. To initialize all members together, the Syntax is: var dog Dog = Dog{Name: "Kenai", Coat: "coat", Eyes: "blue", Breed: "Lab"} Members can be accessed using the (dot) . notation.

Making a Pointer to a struct

A pointer can be created to any variable in go. The general format for doing this is: ptr = &variableName

Our example to the right shows the definition and initialization of a struct as well as creating a pointer to that struct.

Structs and Go's interfaces allow for a form of inheritance and polymorphism. Many people seem to love the setup and boast that is actually better than the classical inheritance found in Java. Interfaces define what a certain type can do. If a type contains the specified methods in the interface, it can be considered the type of the interface. But you don’t explicitly say a data type implements an interface; rather, your data types must implement all of the methods that the interface defines. The other component necessary for inheritance is embedded structs. You can define one type as another type plus some additional features. A problem with the typical class structure is the fragile base problem. If you change the base, it often causes ripple effects from all the classes that inherit from it. Go’s embedded structs encourages composition – that is grouping by features rather than classes. “Go’s struct embedding is amazing. It disguises composition as inheritance and allows easily changing implementation details via its powerful interfaces.”

package main
   
   import (
	   "fmt"
   )
   
   type Dog struct {
	   Name  string
	   Coat  string
	   Eyes  string
	   Breed string
   }
   
   // Parenthesis before toString indicates this function is a
   // METHOD, the purpose of which is to operate on variables of
   // type Dog
   func (dog *Dog) toString() {
	   fmt.Println((*dog).Name, "is a", (*dog).Coat, (*dog).Breed, "with",
		   (*dog).Eyes, "eyes.")
   }
   
   func main() {
	   var adog Dog
	   adog.Name = "Honey"
	   adog.Coat = "brindle"
	   adog.Eyes = "hazel"
	   adog.Breed = "Pitbull"
	   
	   dogPtr := &adog
	   
	   adog.toString()
	   fmt.Println()
	   dogPtr.Name = "Buster"
	   dogPtr.Eyes = "Brown"
	   dogPtr.toString()
   }
